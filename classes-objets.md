# Diagramme de classes et d'objets

## Diagramme de classe
### 3 niveaux:
- Conceptuel
	- le projet entier
	- rôles et responsabilités
- Spécification
	- composants du logiciel
	- interfaces indépendantes
- Mise en oeuvre
	- classes en code (Java, C)
	- sert de schéma pour le code
